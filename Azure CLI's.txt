# NSG?
Network Security Group which actually block and allows the traffic

# RDP?
RDP stands for Remote Desktop Protocol. It is a proprietary technology developed by Microsoft that allows a user to connect to and control another computer or server remotely over a network connection. It provides a graphical interface for working on a remote Windows system.

# VHD?
A Virtual Hard Disk (VHD) is a file format that acts as a container, mimicking a physical hard drive to store data, operating systems, and applications within a virtual machine (VM).

# Docker?
Docker is a platform used to package an application and everything it needs to run (code, libraries, and settings) into a single unit called a container. In simple terms, Docker creates the containers, and Kubernetes manages them at scale.

# Kubernetes?
Kubernetes (often called K8s) is an orchestration platform used to automate the deployment, scaling, and management of containerized applications. 

# Reference VM
A reference VM (or reference machine) is a master virtual machine used as a template to create a base image for deploying multiple, identical virtual machines. It is fully configured with a clean OS install, software applications, and security updates, then captured to serve as a "golden image" for consistent, automated IT deployment.

# AVD?
Azure Virtual Desktop (AVD) is a comprehensive, cloud-based desktop and app virtualization service hosted on Microsoft Azure. It provides secure, remote access to Windows 10/11 desktops and applications from any device, allowing organizations to manage virtual environments centrally without maintaining on-premises infrastructure. 

# Azure App Service?
Azure App Service is a fully managed "Platform as a Service" (PaaS) used for building, deploying, and scaling web apps and APIs. It handles the underlying infrastructure, allowing you to focus on writing your code rather than managing servers, patching operating systems, or configuring web servers.

# App Service Environment?
An Azure App Service Environment (ASE) is a fully isolated, dedicated hosting environment for running Azure App Service apps—including web apps, APIs, and functions—at high scale. It offers superior network security by deploying directly into a user's Azure Virtual Network (VNet), making it ideal for high-security, high-traffic, and enterprise-grade applications requiring dedicated computing power.
