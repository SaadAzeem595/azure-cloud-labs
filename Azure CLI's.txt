# Using Azure CLI (Recommended for DevOps Skills)

Step 1: Open terminal
Step 2: Login
az login
Step 3: Create Resource Group
az group create \
  --name rg-azure-labs \
  --location eastus

# Check regions using CLI:

az account list-locations --output table

# To install the IIS Web Server role along with its management tools and allow for an automatic restart, use this:

Install-WindowsFeature -Name Web-Server -IncludeManagementTools -Restart

# NSG?
Network Security Group which actually block and allows the traffic

# RDP?
RDP stands for Remote Desktop Protocol. It is a proprietary technology developed by Microsoft that allows a user to connect to and control another computer or server remotely over a network connection. It provides a graphical interface for working on a remote Windows system.
